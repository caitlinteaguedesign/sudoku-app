# Sudoku Checker Front-End

This is the front-end for the Sudoku Checker App.

## Development

In this directory, you can run:

### `npm start`

Runs the app in the development mode.

### `npm run build`

Builds the app for production to the `build` folder.

### `npm run serve`

Runs the build version of the app locally. Will not pull data correctly if the server is not also running.

### `npm run clean`

Deletes `node_modules` and `build` folders and their contents (Windows CLI). Helpful for running a fresh install of dependencies, `npm i` or `npm ci` if a `package-lock.json` is present.

## Background

This project was originally bootstrapped with [Create React App](https://github.com/facebook/create-react-app) in 2021. Unfortunately, CRA end of life was also in 2021. In 2026, this project was updated to build with [Vite v3](https://v3.vite.dev/) to address numerous security concerns with `react-scripts` and its dependencies. Started with version 3 for best compatibility with old code and engine requirements.

To learn React, check out the [React documentation](https://reactjs.org/).
